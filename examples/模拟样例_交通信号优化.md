### 1. 问题重述

随着城市机动车保有量持续增长，交通拥堵已成为普遍性难题。（此处省略120字展开）的核心问题是：如何设计交叉口信号灯配时方案，使车辆延误最小、停车次数最少、通行能力最大？（此处省略80字总结）

### 2. 模型假设与符号说明

#### 2.1 模型假设

本文做出以下假设：

1. 车辆到达服从泊松分布；
2. 饱和流率为定值；
3. 每个周期内各相位绿灯时间之和等于周期时长；
4. 不计行人干扰。

#### 2.2 符号说明

| 符号 | 含义 | 单位 |
|------|------|------|
| C | 信号周期长度 | s |
| g_i | 相位i的绿灯时间 | s |
| q_i | 相位i的车辆到达率 | veh/s |
| s_i | 相位i的饱和流率 | veh/s |
| d_i | 相位i的平均延误 | s |

---

### 3. 模型建立与求解

#### 3.1 交通信号优化模型

目标函数为最小化平均延误：

d = Σ(0.5 * C * (1 - g_i/C)^2 / (1 - q_i/s_i)) / n

约束条件包括：
- 各相位绿灯时间满足最小绿灯要求：g_i ≥ g_min
- 周期长度约束：C_min ≤ C ≤ C_max

#### 3.2 改进遗传算法设计

采用实数编码，适应度函数为延误的倒数。改进之处在于：
- 自适应交叉概率：P_c = k1 * (f_max - f') / (f_max - f_avg)
- 自适应变异概率：P_m = k2 * (f_max - f) / (f_max - f_avg)

其中 f_max 为群体最大适应度，f_avg 为平均适应度，f' 为交叉个体中较大适应度，f 为变异个体适应度。

算法流程：
1. 初始化种群（规模50）
2. 计算适应度
3. 选择操作（轮盘赌）
4. 自适应交叉
5. 自适应变异
6. 判断终止条件（迭代200代或连续10代无改进）
7. 输出最优配时方案

### 5. 模型评价与推广

**优点**：模型考虑了多目标优化的实际需求，改进遗传算法收敛速度快，仿真验证充分。
