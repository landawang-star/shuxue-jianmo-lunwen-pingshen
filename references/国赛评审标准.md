# 中国大学生数学建模竞赛（CUMCM）国奖评审标准

> 本文件整理自全国大学生数学建模竞赛历年评审细则，供评审参考。用户可提供自定义评审标准覆盖此文件。

---

## 一、评审总则

全国大学生数学建模竞赛评审遵循"假设的合理性、建模的创造性、结果的正确性、表述的清晰性"四项基本原则。

国奖（全国一等奖、二等奖）论文需在上述四个维度均达到优秀水平，并具备以下特征：
- 模型方法有显著的创新性或改进
- 问题分析深入全面
- 结果数据翔实可靠
- 论文表述规范清晰
- 有独立的模型检验与评价

---

## 二、分项评审维度与权重

| 评审维度 | 权重 | 国奖要求 |
|---------|------|---------|
| 模型假设的合理性 | 20% | 假设有充分依据，与模型方法逻辑一致，在求解过程中得到正确使用 |
| 模型建立的创造性 | 25% | 模型构建有针对性创新或改进，综合运用多种方法，选择有据可依 |
| 结果表述的清晰性 | 25% | 图表清晰规范，结论有定量数据支撑，结果解读直观明确 |
| 论文格式的规范性 | 15% | 结构完整（含摘要/问题重述/模型建立/求解分析/检验评价/参考文献），排版规范统一 |
| 参考文献与引用 | 15% | 文献不少于8篇，覆盖相关领域关键文献，引用标记与文末列表一一对应 |

---

## 三、论文结构要求

国奖论文必须包含以下章节：

1. **摘要**（300-500字）：问题简述、方法概述、主要结果、结论
2. **问题重述**：用自己的语言重述赛题，明确需要解决的核心问题
3. **模型假设与符号说明**：列出所有假设及其依据，统一说明全文符号
4. **模型建立与求解**：模型的数学表达、求解方法、算法流程
5. **模型检验与分析**：灵敏度分析、误差分析、对比验证
6. **模型评价与推广**：模型优缺点、适用范围、改进方向
7. **参考文献**：GB/T 7714 格式，不少于8篇
8. **附录**（可选）：代码、数据表格、推导过程

---

## 四、格式规范要求

| 规范项 | 要求 |
|-------|------|
| 标题编号 | 统一使用阿拉伯数字层级（1、1.1、1.1.1），编号不跳级 |
| 图表编号 | 图按序编号（图1、图2），表按序编号（表1、表2），正文中必须引用 |
| 公式编号 | 公式居中，编号右对齐放在圆括号中，按序编号 |
| 变量说明 | 所有变量首次出现时需明确定义 |
| 参考文献 | GB/T 7714 格式，文中引用使用方括号编号 |
| 页面设置 | A4纸，页边距上下2.54cm左右3.18cm，正文小四号宋体 |
| 页眉页脚 | 页眉标注"全国大学生数学建模竞赛"，页码连续 |

## 五、评分等级参考

| 等级 | 总分区间 | 说明 |
|------|---------|------|
| 国奖一等奖水平 | 90-100 | 五项均优秀，模型有明显创新，论文表现突出 |
| 国奖二等奖水平 | 80-89 | 五项中四项优秀，论文整体质量高 |
| 省奖水平 | 70-79 | 合格，但存在明显可改进之处 |
| 基本合格 | 60-69 | 基本完成题目要求，但研究深度或规范性不足 |
| 不合格 | <60 | 存在重大缺陷或未完成题目基本要求 |

> **注意**：以上为通用参考标准，具体评审以当年赛题和评审委员会要求为准。本 Skill 输出的是达标程度分析，不作为最终获奖判定。
