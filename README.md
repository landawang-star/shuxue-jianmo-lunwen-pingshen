# 数学建模国赛论文评审助手

> 按照中国大学生数学建模竞赛（CUMCM）国奖评审标准，对参赛论文进行格式规范性、模型假设合理性、建模创造性、结果清晰度、参考文献引用的全面审核，输出分项评分表与修改建议报告的 Skill。

## 简介

本 Skill 帮助数学建模竞赛评委、指导教师和参赛学生，按照中国大学生数学建模竞赛（CUMCM）国奖评审标准，对参赛论文进行全面审核，输出结构化评审报告与修改建议。

## 功能特性

- 📄 **论文结构与格式审核**：章节完整度、图表编号、公式规范、参考文献格式、摘要规范、页面规范
- 🎯 **五维度内容评审**：
  - 模型假设的合理性（20%）
  - 模型建立的创造性（25%）
  - 结果表述的清晰性（25%）
  - 论文格式的规范性（15%）
  - 参考文献与引用（15%）
- 📊 **分项评分表**：每项 0-20 分，总分换算为 0-100 分，含评分理由
- 📝 **格式问题清单**：维度 + 问题描述 + 位置 + 严重程度 + 修复建议
- 🚀 **修改提升建议**：按 P0/P1/P2 优先级排序的可操作建议

## 适用场景

- 数学建模竞赛评委批量化审核参赛论文
- 指导教师对学生作品进行赛前预审
- 参赛团队自查论文是否达到国奖标准
